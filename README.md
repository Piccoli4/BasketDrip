### Instalación 
#### List
- Clonar repositorio
    - git clone https://github.com/Piccoli4/BasketDrip
- Moverse a la carpeta del proyecto.
    - cd ecommerce
- Instalamos las dependencias
    - npm install
- Compilamos
    - npm run dev
